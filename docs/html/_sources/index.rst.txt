rt2_ass1
========
Documentation of the ROS package rt1_ass2 which implements the requirements for assignment 2 of
the course Research Track 1 of Robotics Engineering course by University degli Studi di Genova.
The assignment is based on the ROS package assagnemnt_2_2022 which implements, among other features,
an action server to drive a robot with the Bug 0 algorithm through a predefined environment.
